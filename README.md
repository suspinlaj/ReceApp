# 🍔 **ReceApp**

**ReceApp** es un recetario interactivo y visualmente muy cuidado que no solo te enseña cómo preparar distintos platos, sino que te ayuda a decidir qué cocinar basándose en los ingredientes que ya tienes en la nevera. 

## 📘 **Acerca de este proyecto**

Este proyecto fue desarrollado como ejercicio práctico para la asignatura de **Programación multimedia y de dispositivos móviles**. El objetivo era crear una aplicación de gestión de datos, pero quise darle un enfoque muy centrado en la Experiencia de Usuario (UX) y el diseño de Interfaz (UI). Cuenta con una paleta de colores pastel, tipografías amigables y un diseño de fondo personalizado que hace que la navegación sea intuitiva y agradable.

## 🔧 **Funcionalidades**

- **Inventario de Ingredientes**: Una interfaz interactiva donde el usuario puede marcar mediante *checkboxes* qué ingredientes tiene actualmente disponibles en su cocina.
- **Algoritmo de Búsqueda Inteligente**: La aplicación cruza los datos de los ingredientes seleccionados con la base de datos de recetas para calcular la viabilidad de cada plato.
- **Feedback Visual de Progreso**: La lista de resultados es dinámica. Muestra una barra de progreso con el porcentaje exacto de ingredientes que posees para cada receta y cambia el color del título según tu disponibilidad:
  - 🟢 **Verde:** Tienes el 100% de los ingredientes (¡Lista para cocinar!).
  - 🔴 **Rojo:** Te faltan algunos ingredientes.
  - 🟤 **Marrón:** No tienes los ingredientes necesarios.
- **Detalle de Receta**: Pantalla individual para cada plato que muestra la lista completa de ingredientes necesarios y los pasos de preparación detallados.

## 💻 **Tecnologías y Entorno**

- **Lenguaje**: Kotlin
- **Entorno de pruebas / Emulador**: Medium Phone
- **Enfoque destacado**: Lógica de filtrado de datos y actualización dinámica de UI.

## ⚙️ **Estado del proyecto**

Finalizado. Proyecto educativo completado con la lógica de cálculo de ingredientes y todo el diseño visual implementado.
